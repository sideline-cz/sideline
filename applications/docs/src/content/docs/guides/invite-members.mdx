Invites are how new people join your team. Each invite is a URL that, when opened, signs the user in with Discord and adds them to your roster. You can have multiple active invites at the same time — for example, one invite per group.

## Generating an invite

<Steps>

1. Open your team → **Roster** → **Invites**.
2. Click **New invite**.
3. Optionally scope to a **group** — the new member is automatically added to that group on join.
4. Optionally set an **expiry date**.
5. Copy the link and share it.

</Steps>

<Aside type="tip" title="Group-targeted invites">
  Create one invite per group (First Team, Reserves, Juniors…) and share the right link with the right people. Each new member lands in the correct group automatically, and the bot's welcome message fills in the group name.
</Aside>

## What the new member sees

<Steps>

1. They click the invite URL.
2. Sideline shows the team name, which group they'll join, and who invited them.
3. They sign in with Discord.
4. They are added to your team (and the target group if the invite is group-scoped).
5. The page briefly shows a **"Preparing your Discord invite…"** state while the bot generates a personalised, single-use Discord invite link. Within about a second, the browser redirects them straight to the Discord server.

</Steps>

<Aside type="note" title="Re-connect Discord prompt">
  If a member signed in to Sideline before group-targeted invites were introduced, the invite page may show a **"Re-connect Discord"** button instead of completing immediately. The member should click it — Sideline re-requests the necessary permission and finishes the Discord server join automatically. This is a one-time step.
</Aside>

<Aside type="note" title="How the Discord link works">
  Each member who clicks "Accept" receives their own unique Discord invite link. The link is valid for 24 hours and can be used once. There is no shared Discord server link for captains to copy — share only the `/invite/{code}` web URL with your players.
</Aside>

## Discord welcome message

If you have configured a **welcome channel** and a **welcome message template** in Team settings, the bot posts a welcome embed in that channel when the new member joins the Discord server. The embed uses the invite to fill in the inviter's name and the group name. See [Discord integration](/guides/discord-integration/#welcome-message) for the full placeholder reference.

## Deactivating an invite

<Aside type="tip" title="Deactivate quickly">
  If an invite leaks or you shared the wrong one, open **Roster → Invites**, find the invite in the list, and click **Deactivate**. Any future attempt to use that link will fail. Other active invites are not affected.
</Aside>
