Reminders are Sideline's most-loved feature. They make sure nobody has to manually chase players for RSVPs.

## When reminders fire

Sideline sends one reminder per event, at a configurable time of day on a configurable number of days before the event. For example, the default configuration fires at **18:00** on the day that is **1 day before** the event, evaluated in the team's configured timezone (default: `Europe/Prague`).

The exact timing is configurable per team from **Team settings → Notifications**:

| Setting | What it controls |
|---|---|
| **Days before** | How many days before the event start the reminder fires. Set to `0` to disable reminders entirely. |
| **Time of day** | The local time (in the team's timezone) at which the reminder fires. Must be 23:54 or earlier. |
| **Timezone** | An IANA timezone name (e.g. `Europe/Prague`, `America/New_York`) used to evaluate the reminder time. |
| **Reminders channel** | The Discord channel where reminders are posted. If unset, the event owner-group's channel is used as a fallback. |

## Where reminders go

- **In Discord**, in the team's configured reminders channel (or the event's owner-group channel if no dedicated reminders channel is set).
- Reminders are **one-off posts** — once the reminder is sent, the event is marked so it won't fire again.

## Who gets pinged

Only players who are members of the event's **member group** (and any of its sub-groups) who haven't RSVPed yet. Once someone clicks Yes / No / Coming later, they drop off the reminder list.

The bot sends a direct message to each non-responder who has a linked Discord account, with a link to the event's voting message.

## Event-start announcements

When an event reaches its start time, the bot posts a fresh **"Starting now: \{title\}"** message to the same reminders channel. This announcement:

- Lists everyone who said **Going**, filtered to the event's member group.
- @-mentions the member group's Discord role.
- Is separate from the in-place edit of the original event embed (which simply removes the RSVP buttons).

## Disabling reminders

Per-team: from **Team settings → Notifications**, set **Days before** to `0`.

<Aside type="tip" title="Reminder hygiene">
  Reminders are effective when they're rare enough to matter. One well-timed reminder the day before is more effective than multiple noisy pings. Trust the defaults.
</Aside>
